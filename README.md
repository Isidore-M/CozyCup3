CozyCup is a modern, high-end coffee shop application built with Jetpack Compose and Kotlin. It is designed to provide users with a seamless, "cozy" experience for browsing premium coffee products and making secure payments.

‚ú® Key Features**

--------------------------------------------------------------------------------------------------------------------------

Product Detail Screens: Immersive and interactive UI built with Jetpack Compose to showcase coffee origin, notes, and pricing.

Secure Checkout: Integration with Stripe payment methods for a safe and reliable purchasing experience.

Modern UI/UX: Designed with a focus on clean typography and smooth transitions to reflect a premium brand identity.

Reactive UI: Fully built using a declarative approach with Jetpack Compose for a responsive user experience.
